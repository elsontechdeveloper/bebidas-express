# Bebidas Express - Landing Page

## Como usar:

### Opção 1: Compilar SCSS online (Recomendado para iniciantes)
1. Acesse um compilador SCSS online como:
   - https://sass-lang.com/install
   - https://jsonformatter.org/scss-to-css
   - https://codebeautify.org/scss-to-css-converter

2. Copie todo o conteúdo do arquivo `scss/styles.scss`
3. Cole no compilador online
4. Copie o CSS gerado e cole em `css/styles.css`

### Opção 2: Usar Node.js/Sass (Recomendado para desenvolvedores)
1. Instale o Sass: `npm install -g sass`
2. No terminal, navegue até a pasta do projeto
3. Execute: `sass scss/styles.scss css/styles.css`
4. Para watch mode (atualização automática): `sass --watch scss/styles.scss css/styles.css`

### Opção 3: VS Code com Live Sass Compiler
1. Instale a extensão "Live Sass Compiler" no VS Code
2. Abra a pasta do projeto
3. Clique em "Watch Sass" na barra inferior
4. O CSS será gerado automaticamente

## Abrir o projeto:
Basta abrir o arquivo `index.html` no seu navegador.

## Estrutura:
- index.html - Estrutura HTML
- scss/styles.scss - Arquivo SCSS com variáveis, mixins e estilos
- css/styles.css - CSS compilado (gerado automaticamente)
